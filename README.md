## POS Awesome


#### How to Install

1. `bench get-app  git@github.com:yrestom/POS-Awesome.git`

2. `bench setup requirements`

3. `bench build --app posawesome`

4. `bench restart`

5. `bench --site [your.site.name] install-app posawesome`

6. `bench --site [your.site.name] migrate`
